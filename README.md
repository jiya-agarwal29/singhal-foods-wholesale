# singhal-foods-wholesale
Full-stack MERN wholesale grocery ordering platform for hotels, restaurants, and retailers.
## 🚀 Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- JWT Authentication

## 🎯 Project Goal

Build and deploy a production-like wholesale ordering platform for Singhal Foods.

## 📌 Planned Features

### Customer
- Product Catalog
- Search Products
- Categories
- Shopping Cart
- Login & Register
- Place Orders
- Order History

### Admin
- Dashboard
- Product Management
- Stock Management
- Order Management
- Low Stock Alerts
